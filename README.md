# HappyMeals

Приложение на C++ c фреймворком Qt и Docker для подбора блюд на основе имеющихся ингредиентов.

## Описание
Программа помогает пользователю определиться, что приготовить, исходя из того, что есть в холодильнике.

## Функции
- Поиск рецептов по списку продуктов.
- Фильтрация блюд по категориям (завтрак, обед, ужин).
...

## Технологии
- Язык: C++11
- Компилятор: g++
- Фреймворк: Qt
